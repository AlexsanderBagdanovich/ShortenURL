# URL Shortener (.NET 8)

## Требования
- .NET 8
- MySQL или MariaDB (локально или на сервере)
- Любой текстовый редактор / IDE (VS, Rider, VS Code)

## Настройка
1. Откройте appsettings.json и укажите свои данные MySQL/MariaDB
- Важно: Убедитесь, что база данных MySQL запущена, а указанный пользователь имеет права на создание базы и таблиц.

## Запуск проекта
1. Откройте терминал в корне проекта.
2. Перейдите в папку API (ShortenURL.Api):
3. Запустите проект.

## Использование
- Создать ссылку: введите URL и нажмите "Create".
- Редактировать ссылку: кнопка “Edit” рядом с нужной ссылкой.
- Удалить ссылку: кнопка “Delete”, с подтверждением.
- Переход по короткой ссылке: используйте ссылку вида http://localhost:5000/r/abc123.
- Количество кликов будет автоматически увеличиваться при каждом переходе.

## Примечания
- PRG pattern используется для формы Create/Edit, чтобы избежать повторной отправки данных при обновлении страницы.
- IncrementClicksAsync делает обновление количества кликов максимально быстро через SQL, не загружая всю сущность.
- Валидация URL гарантирует, что создаются только корректные HTTP/HTTPS ссылки.
- Все зависимости настроены через DI (DependencyInjection.cs) — проект готов к тестированию и расширению.
